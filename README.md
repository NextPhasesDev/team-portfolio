## About This Repository

This is the source code for the NextPhases website. The site is built with clean, modern technology and is production-ready.

### Built With

- **HTML5 & CSS3** - Semantic, responsive markup
- **Vanilla JavaScript** - No frameworks, lightweight & fast
- **Modern UI/UX** - Dark/light mode, smooth animations, accessibility-first
- **Form Backend** - Formspree integration for client inquiries
- **Deployment** - Netlify with automatic SSL

### Key Features

- 📱 Fully responsive across all devices
- 🌓 Dark and light theme support
- 🌍 Multi-region pricing (auto-detects currency)
- ♿ Accessible design (WCAG compliant)
- ⚡ Fast load times with no JavaScript framework dependencies
- 🔒 Privacy-focused, secure form handling
- 🌐 SEO optimized with sitemap & structured data
